Proyectos Desarrollados

WIQEII

La aplicación web WIQ permite a los usuarios jugar un juego similar al del concurso Saber y Ganar.
Este juego consiste en responder a una serie de preguntas con diferentes tipos y temas obteniendo un premio para cada pregunta bien respondida. Las preguntas del juego se generan automáticamente a partir de los datos disponibles en Wikidata

SaveUp

SaveUp es una aplicación móvil para Android cuya función principal es la de ayudar al usuario a gestionar sus ahorros y transacciones

La aplicación además incluye varias funcionalidades como la de establecer límites de gasto mensuales, metas de ahorro, creación y gestión de grupos para compartir transacciones, etc.
